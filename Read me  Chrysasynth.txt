 🎛️ Chrysasynth

Chrysasynth is a web-based musical instrument designed to explore grooves, basslines and melodies through an intuitive and immersive interface.

It is not a traditional DAW, but a focused creative tool built to encourage fast experimentation and musical flow.

---

✨ Overview

Chrysasynth provides a minimal yet expressive environment where rhythm, harmony and texture evolve in real time.

The system combines structured sequencing with controlled variation to create a feeling of a "living track", without breaking musical coherence.

---

 ⚙️ Features

- Drum sequencing (Kick, Snare, Hat, Clap)
- Bass engine with A/B pattern logic
- Melody generator with controlled variation
- Auto Evolve system (subtle musical transformations)
- Session saving (Seeds)
- Real-time parameter control (BPM, Swing, Scale)
- Visual feedback and interactive UI

---

 🎧 Philosophy

Chrysasynth is designed around a simple idea:

> Music creation should be immediate, fluid, and inspiring.

Instead of overwhelming the user with options, it focuses on:
- clarity
- responsiveness
- musical coherence

---

 💻 Platform
